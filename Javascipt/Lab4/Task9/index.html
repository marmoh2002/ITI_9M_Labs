Summary:

both should display the raw html tags as text, not render them as bold

  • textContent returns everything including the hidden span innerText ignores elements with display:none
  • textContent keeps all the whitespace as-is innerText kind of normalizes it like how the browser renders it
  • textContent is supposedly faster because it doesnt care about styling or layout stuff
  • innerText wont work on elements that arent rendered at all
